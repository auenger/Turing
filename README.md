# 图凌人工智能官网

图凌人工智能官方网站，使用 Astro 及 Tailwind CSS 构建。

## 本地开发

```sh
pnpm install
pnpm astro dev --background
```

后台服务可通过以下命令管理：

```sh
pnpm astro dev status
pnpm astro dev logs
pnpm astro dev stop
```

## 构建

```sh
pnpm build
```

## GitHub Pages
